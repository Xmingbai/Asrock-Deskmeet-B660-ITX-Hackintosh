# 本机配置清单

主机： asrock Deskmeet B660 itx

CPU: i5-12490F

内存： 32G 2x16 DDR4 3200

硬盘：SN750（Mac）和 PM9A1 500G（Windows）

散热： AXP90-X53 黑化版

显卡： RX5500XT 4G

网卡：BCM94360CS2+转接卡



# 完善程度

CPU 睿频正常 12490F性能相当于i9-10900

1G有线网卡支持；

音频前后输出正常；

睡眠与唤醒正常，支持USB设备唤醒

隔空投送、通用控制支持


BIOS 设置参考：https://www.bilibili.com/read/cv12293964
